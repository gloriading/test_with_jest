## JEST

* Run JEST through command line
1. Globally install JEST 

```
npm install jest --global
```
2. Run all JEST tests
```
jest
```
3. Run one test file
```
jest my-test
or
jest path/to/my-test.js
```
4. Watch Mode
```
jest --watch
```
